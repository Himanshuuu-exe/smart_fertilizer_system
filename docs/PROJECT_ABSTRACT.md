## Abstract

Agriculture plays a vital role in the Indian economy, and efficient fertilizer management is crucial for sustainable crop production. This project presents a **Smart Fertilizer Recommendation System** — an AI-powered web application that leverages machine learning to provide personalized fertilizer recommendations based on soil nutrient levels and environmental conditions.

The system accepts eight input parameters: **Nitrogen (N)**, **Phosphorous (P)**, **Potassium (K)**, **Temperature**, **Humidity**, **Moisture**, **Soil Type**, and **Crop Type**. Using a trained **Random Forest Classifier** — selected from a comparison of five algorithms — the system recommends the most suitable fertilizer with a confidence percentage.

The application was developed as a full-stack project:
- **Frontend**: React.js with Vite and Tailwind CSS — featuring glassmorphism UI, dark/light mode, animated charts, and responsive design
- **Backend**: Python FastAPI REST API — providing prediction, health, and metrics endpoints
- **Machine Learning**: Scikit-learn — Random Forest achieving **95.2% accuracy** with cross-validation

Key deliverables include: a 2,200-sample dataset, trained ML model with encoders, interactive web UI with nutrient analysis and farming tips, prediction history, analytics dashboard, and deployment-ready configuration for Render.

**Keywords**: Machine Learning, Fertilizer Recommendation, Random Forest, FastAPI, React, Precision Agriculture, Soil Analysis
